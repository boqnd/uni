Текст на въпроса

Напишете функция за таблицата FN7XXXX.EMPLOYEE (където FN7XXXX е схемата с вашия факултетен номер), която по номер на служител връща конкатенирани имената на служителя с един интервал между тях. Напишете заявка, която извиква функцията с подходяща стойност.

Копирайте скрипта като отговор на задачата.


Answer text

CREATE FUNCTION CONCAT_NAMES(V_EMPNO CHARACTER(6))
RETURNS VARCHAR(32)
RETURN
SELECT (EMPLOYEE.FIRSTNME CONCAT ' ' CONCAT EMPLOYEE.LASTNAME)
FROM FN72061.EMPLOYEE
WHERE EMPNO = V_EMPNO;

SELECT EMPNO, CONCAT_NAMES(EMPNO)
FROM FN72061.EMPLOYEE;

Текст на въпроса

Напишете изглед за таблицата FN7XXXX.EMPLOYEE (където FN7XXXX е схемата с вашия факултетен номер), който да извежда номер на отдел и броя на служителите на длъжност MANAGER или PRES за този отдел. Напишете заявка, която извиква изгледа.

Копирайте скрипта като отговор на задачата.


Answer text

CREATE VIEW FN72061_VIEW1(WORKDEPT_NUMBER, COUNT_JOBS)
AS SELECT WORKDEPT, COUNT(*)
FROM FN72061.EMPLOYEE
WHERE JOB = 'MANAGER' OR JOB = 'PRES'
GROUP BY WORKDEPT;

SELECT * FROM FN72061.FN72061_VIEW1;

От максимално 1,00

Текст на въпроса

Напишете тригер за таблицата FN7XXXX.EMPLOYEE (където FN7XXXX е схемата с вашия факултетен номер), който при обновяване на отдела на служител проверява дали той е MANAGER и ако е не позволява да бъде обновен. Напишете заявка, която тества тригера.

Копирайте скрипта като отговор на задачата.


Answer text

CREATE TRIGGER CHANGE_DEPT

INSTEAD OF UPDATE ON FN72061.EMPLOYEE
REFERENCING NEW AS NEWEMP OLD AS OLDEMP
FOR EACH ROW
    WHEN ( OLDEMP.JOB <> 'MANAGER' )
        UPDATE FN72061.EMPLOYEE
        SET(FIRSTNME,MIDINIT,LASTNAME,WORKDEPT, EDLEVEL, HIREDATE)
                = (NEWEMP.FIRSTNME, NEWEMP.MIDINIT, NEWEMP.LASTNAME, NEWEMP.WORKDEPT, NEWEMP.EDLEVEL, NEWEMP.HIREDATE)
        WHERE EMPNO = OLDEMP.EMPNO;
